## KR — 여긴 작업장이 아니라 놀이터다

본계정 [`hkimw`](https://github.com/hkimw)가 정장 입고 회의실 들어가는 계정이라면, 여기는 헬멧 쓴 유령이 비커 들고 뛰어다니는 **실험실 놀이터**다.

깔끔한 포트폴리오? 그런 건 위층에 있음.  
여기는 `아이디어 → 폭주 → README → 배지 추가 → 또 폭주` 순서로 굴러간다.

- 만들다 만 것
- 만들다 너무 커진 것
- 본계정에 올리기엔 아직 거친 것
- 과제에서 출발했는데 갑자기 엔진 달린 것
- 새벽 2시에 시작해서 아침에 왜 돌아가는지 모르는 것
- 코드보다 README가 먼저 이상해진 것

> 목적: 완성도 높은 척하기 ❌  
> 목적: 이상한 걸 끝까지 밀어붙여서 살아남는 놈 찾기 ✅

## EN — Not a portfolio. A playground with stickers.

If [`hkimw`](https://github.com/hkimw) is the account that walks into the room wearing a suit, this one is the ghost in a hardhat running through the hallway with a glowing beaker.

This is not the polished shelf.  
This is the **personal engineering playground** where strange ideas get thrown into a container, given a README, shaken gently but enthusiastically, and observed from a safe distance.

It contains:

- unfinished things
- oversized side quests
- coursework that escaped the folder
- prototypes too rough for the main account
- demos that work only because the machine is afraid of them
- documentation that started normal and became a control panel

> Goal: pretend everything is clean ❌  
> Goal: push weird ideas until the surviving ones become useful ✅
